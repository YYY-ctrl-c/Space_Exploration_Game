# 星际探险家：队员疲劳管理的星球探索系统
## 一级目录功能说明
1. **src/**
   
   源代码主目录，包括：
   
   ​	frontend：前端页面（Vue3）

   ​	backend：后端接口（Java Servlet）
   
   ​	database：数据库SQL脚本
   
2. **doc/**

   课程设计文档，项目截图、测试报告

3. **test/**

   功能测试用例，接口测试记录

4. **README.md**

   项目说明文档，目录结构与功能介绍

## 技术栈
- 前端：Vue3 
- 后端：Java Servlet
- 数据库：MySQL 8.0
- 架构：前后端分离 B/S 架构

## 核心功能模块
1. 队员管理（疲劳度、属性、状态）
2. 星球探索（消耗疲劳、随机奖励、事件）
3. 队员招募（抽卡系统）
4. 图鉴系统（队员、星球、资源）
5. 商店系统（物资、装备、恢复道具）
6. 用户登录注册
